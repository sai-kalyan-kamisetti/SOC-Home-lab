# SOC-Home-lab
SOC home lab using VirtualBox, Kali Linux, Ubuntu instrumented with wazuh server &amp; Windows VM instrumented with Sysmon + Splunk Universal Forwarder and wazuh agent. Simulates brute force, port scanning, Metasploit exploits &amp; persistence attacks — detected and analyzed via Splunk Enterprise SIEM running on macOS.
